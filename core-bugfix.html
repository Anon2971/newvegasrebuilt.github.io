A Note About Load Order and Conflict Resolution

LOOT is not used in this guide. The best way to sort your load order is to use FNVEdit to view conflicts and manually sort your load order based off that and then create a custom conflict resolution patch. The guide will provide a fully-modular handmade conflict resolution FOMOD along with the proper load order for each module. This means that any mods listed can be left out without missing out on the pre-made conflict resolution/load order. However, if you plan on adding any big mods, you will need to familiarize yourself with manually solving conflicts in xEdit. I suggest this tutorial for familiarizing yourself with the tool and this for making custom conflict resolution.

A Note About the Plugin Limit

Fallout New Vegas has a safe plugin cap of about ~120 (no one knows the exact number) in vanilla, but the Mod Limit Fix in Core - Utilities raises that to 255. However, it is still recommended to try to stay below ~120 plugins for optimal performance/load times. The guide will use the tool zMerge (a.k.a zEdit) to merge plugins together to stay below the limit. If you plan on adding your own mods in addition to the ones in the guide, familiarize yourself with the tool so you can stay below the limit as well.
