A Note About Testing the Game

After each section in every module, you should launch the game and make sure everything is working correctly by running in-game around for a few minutes

xNVSE

Installation Instructions:

  • Download the most recent version at the top of the page
  • Extract all files ending in .dll and the .exe (4 total files) from the archive to the Root folder
    • nvse_steam_loader.dll is still needed with the GOG version of the game

NOTE: If you do not know what the Root folder is, go to the Core - Installation page and read the Common Terminology section

- An updated version of NVSE, which extends the scripting capabilities of the engine and is essential for most mods

NVSE_Config.ini

Installation Instructions:

  • Extract the nvse folder from the archive to the game's Data folder

NOTE: If you do not know what the Data folder is, go to the Core - Installation page and read the Common Terminology section

- Increases heap allocation to help prevent stuttering/crashing

JIP LN NVSE Plugin

Installation Instructions:

  • Extract the nvse folder from the archive to the game's Data folder
  • The files from the mod, and the rest of the NVSE plugins in this section, should end up like this (pay attention to the filepath at the top)

NOTE: The reason that these NVSE plugins are installed directly into the data folder and not through MO2 is that MO2's virtual file system can (rarely) fail to hook the .DLLs in time when launching the game and therefore they will not work in-game

- NVSE extension with 800+ new functions

4GB Patcher

Installation Instructions:

  • Extract the .exe from the archive to the root folder
  • Right-click on FalloutNVpatch.exe and select Run as administrator
  • When the command prompt appears, hit enter
  • The command prompt should now exit and FalloutNV_backup.exe should appear in the root folder
  • From now on, run the game through the New Vegas option in MO2
    • DO NOT run the game through the Fallout Launcher option, this will reset the Fallout.ini and FalloutPrefs.ini in MO2

- Makes the game large address aware, meaning it can use 4GB of RAM instead of 2GB
NOTE: Verifying your game files through either Steam or GOG will undo the patch, so you must rerun it after
NOTE 2: THIS PATCH IS STILL NEEDED WITH THE GOG VERSION OF THE GAME FOR THE NVSE INTEGRATION

Verifying NVSE and the 4GB Patch are Working

  • Run the game through the New Vegas option in MO2
    • You do not need to use the NVSE option, as the 4GB patch autoloads NVSE when the game is ran through the default executable. And the default executable is used because, only in the GOG version, the NVSE_Config.ini from above will not be loaded if you use run through NVSE
  • Once the game has reached the title screen, open the console by pressing the Tilde (~) key
  • Input getnvseversion into the console and hit enter
    • The console should report NVSE version: 5
      • If it does not, verify NVSE is installed correctly and retry
  • Input getislaa into the console and hit enter
    • The console should report GetisLAA >> 2.0000
      • If the console reports GetisLAA >> 0.0000, the patch is not installed correctly
      • If the console reports GetisLAA >> 1.0000, the patch is installed correctly but is not working correctly, so you must run the game as an administrator
      • If the console reports an error message, you did not install JIP LN NVSE correctly

New Vegas Tick Fix

Installation Instructions:

  • Download the mod manually
  • Extract the nvse folder from the archive to the game's Data folder

Manual editing required:

  • Open NVTF.ini in Fallout New Vegas/Data/nvse/plugins
  • Set bUseD3D9Ex to 1 (line 37)
    • With this setting enabled, you must run the game in exclusive fullscreen mode (the INI tweaks from the previous step already force the game to run in exclusive fullscreen)
  • Set bToggleTripleBuffering to 1 (line 42)
  • Set bUseFlipExSwapMode to 1 (line 60)
  • Set bResizeHashtables to 1 (line 79)
    • NOTE: If you are on Windows 7, you do not need to enable the previous three options
    • Enabling this option will break the in-game screenshot function (Windows print screen will still work), but will allow for safe alt-tabbing in fullscreen. It is fine to leave disabled if you don't alt-tab and want to use in-game screenshots
  • Set iMaxFPSTolerance to 1 less than your monitor's refresh rate (line 86)

- Works to fix micro-stuttering, improve performance, and allow playing at higher framerates
NOTE: If you experience incredibly low frame rates and bad stuttering with this mod, set bUseD3D9Ex to 0. This setting can rarely cause that on some systems

Mod Limit Fix

Installation Instructions:

  • Download the mod manually
  • Extract the NVSE folder from the archive to the game's Data folder

- Raises the game's ~120 plugin limit to 255. May also increase performance and decrease load times, especially with more heavily-modded games
NOTE: Although this does raise the plugin limit, it is still recommended to stay below ~120 plugins for optimal performance/load times
