Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator.
  2. Name the separator Content.

Uncut Wasteland

A custom version of sandbox6's amazing Uncut Wasteland mod, which restores many pieces of cut scenery and a few NPCs that were cut because of console limitations.

Essential DLC Enhancements Merged

Installation instructions:

  • Main File - Essential DLC Enhancements

Merge of many gameplay improvements for the DLCs, all updated and fixed.

Mojave Raiders

Installation instructions:

  • Main File - Mojave Raiders
  • Additionally install Mojave Raiders Tweaks (Install manually in MO2)
  • Additionally install JSawyer Ultimate - Mojave Raiders Patch and Mojave Raiders - Essential Vanilla Enhancements Merged Patch (When prompted, select the name that matches the file on Nexus)

Major overhaul/balancing of the vanilla raiders, as well adding many new spawn points.

The Living Desert

Installation instructions:

  • Main File - The Living Desert - Main File 2.3
  • Additionally install The Living Desert - Essential Vanilla Enhancements Merged Patch (When prompted, select the name that matches the file on Nexus)

Adds hundreds of new NPCs and scripted events that make the world feel much more alive and reactive.
