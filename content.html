Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select All Mods -> Create Separator
  2. Name the separator Content

Uncut Wasteland

Installation instructions:

  • Main File - Uncut Wasteland plus NPCs
  • Optional File - Uncut Extra Collection

- Restores many pieces of cut scenery and a few NPCs that were cut because of console limitations

Mojave Raiders

JSawyer Ultimate Edition - Mojave Raiders Patch

  • Main File - JSawyer Ultimate - Mojave Raiders Patch

Mojave Raiders Tweaks

The Living Desert

- Adds hundreds of new NPCs and scripted events that make the world feel much more alive and reactive

Because of all the new NPCs/scripts this mod adds it can be somewhat performance-intensive in some areas, especially on lower-end systems. If you have a weaker PC, you may want to skip this mod.
