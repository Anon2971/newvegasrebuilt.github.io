The Goal of the Core Module:

The goal of this module is to set up all the essential modding tools for FNV as well as some essential bug fixes/quality of life improvements that give a very similar feel to vanilla, just without all the bugs and oversights. It is intended for anyone with any level of modding experience to be able follow it easily. It is all also fully modular, meaning you can leave out any mods you don't like (the same goes for the other modules as well). All other changes such as graphic overhauls, gameplay mods, and new quests/weapons that you are accustomed to in fully modded setups are covered in the other 3 modules.

Requirements:

  • An English copy of FNV with all the DLC from either Steam or GOG (Instructions for properly installing it are in the Core module)
  • Windows 10 64bit (Windows 7 and 8 64bit will technically work, but are unsupported)
  • A text editor (I recommend Notepad++, but default Windows notepad will work)
  • An archiving tool (I recommend 7-Zip)
  • Latest GPU drivers (Either NVIDIA or AMD)
  • At least 15 gigabytes of free drive space for just the Core module, and 23 gigabytes for the entire guide
  • NOT compatible with DUST / TTW / New California out of the box
    • If you want to use this as a base for either of those mods, you must be aware of the incompatibilities (DUST, TTW and New California)
  • A Nexus account
