Requirements:

  • A copy Fallout New Vegas with all the DLC from either Steam or GOG (There will be instructions for properly installing it in the next step)
  • Windows 7 or newer (64bit)
  • A text editor (I recommend Notepad++)
  • An archiving tool (I recommend 7-Zip)
  • Latest GPU drivers (Either NVIDIA or AMD)
  • NOT fully compatible with TTW/New California out of the box. If you want to use this as a base for either of those mods, you must be aware of the incompatibilities
