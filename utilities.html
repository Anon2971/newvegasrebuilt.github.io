A Note About Testing the Game

After each section in every module, you should launch the game and make sure everything is working correctly by running around for a few minutes and testing
various parts of mods to make sure they are working correctly

Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Utilities into the pop-up and select OK

4GB Patcher

Special install intructions:

  • Extract the .exe from the archive to the root folder
  • Right-click on FalloutNVpatch.exe and select Run as administrator
  • When the command prompt appears, hit enter
  • The command prompt should now exit and FalloutNV_backup.exe should appear in the root folder
  • From now on, run the game through the New Vegas option in MO2

- Makes the game large address aware, meaning it can use 4GB of RAM instead of 2GB
  Verifying your game files through either Steam or GOG will undo the patch, so you must rerun it after

Verifying NVSE and the 4GB Patch are Working

  • Run the game through MO2
  • Once the game has reached the title screen, open the console by pressing the Tilde (~) key
  • Input getnvseversion into the console and hit enter
    • The console should report NVSE version: 5
      • If it does not, verify NVSE is installed correctly and retry
  • Input getislaa into the console and hit enter
    • The console should report GetisLAA >> 2.0000
      • If the console reports GetisLAA >> 0.0000, the patch is not installed correctly
      • If the console reports GetisLAA >> 1.0000, the patch is installed correctly but is not working correctly, so you must run the game as an administrator
      • If the console reports an error message, you did not install JIP LN NVSE correctly

New Vegas Tick Fix

Special install instructions:

  • Download the mod manually
  • Extract the contents of the archive to Fallout New Vegas/Data/nvse/plugins/
    • If this folder does not exist, create it
  • Open FastExitnTickCountReplace.ini
  • Set bFastResolution to 1

- Stripped down version of NVSR designed to specifically work with Windows 10

A Note About NVSR:

You may be wondering why NVSR isn't in the guide. NVSR has many broken settings, that when turned off essentially just turn off the features that remove the stuttering,
making it pointless. There are also bugs with NVSR unrelated to settings, such as the really long fingers and bouncing NPCs. That being said, the following settings are broken in NVSR:

  • bManageFPS: Doesn't straight up limit FPS, but maintains a minimum FPS, so it starts to disable LOD and other things to maintain that FPS (paraphrased from RoyBatty)
  • bHookLightCriticalSections (Only fully broken on Windows 10): Bad implementation, broken by Windows 10 because the author relied on undefined behavior instead of
    following the API. Causes crashes every 15-ish minutes
  • bReplaceHeap: Old implementation, extremely unstable, not large adress aware. Performs in function by increasing FPS, but at the cost of a much higher crash rate

These settings are broken, no ifs, ands, or buts, about it. Don't listen to anyone who tells you otherwise

GECK Extender

Files to install:

  • Extract the nvse folder from the Geck Extender file to the Data folder

  • The command prompt should automatically apply the patch and allow you to exit, and Geck_backup.exe should appear in the root folder

- Extends GECK functionality and fixes many bugs

BSA Decompressor

Special install instructions:

  • Do not install if you have your game on a slow HDD and/or are using/plan to use TTW (TTW is not a part of the guide)
  • Extract the downloaded archive to your modding folder
  • Right-click FNV BSA Decompressor.exe and select Run as administrator
  • Read and follow the instructions of the installer carefully
  • Open MO2 and click the button on the top bar and select INI Editor
  • Find the following line in the fallout.ini tab
    • SArchiveList=
  • Replace the entire line with the following (should all be on one line):
    • SArchiveList=Fallout - Invalidation.bsa, Fallout - Textures.bsa, Fallout - Textures2.bsa, Fallout - Meshes.bsa, Fallout - Meshes2.bsa, Fallout - Voices1.bsa, Fallout - Sound.bsa, Fallout - Misc.bsa

- Decompresses BSAs and repacks them without zlib compression and fixes certain audio files that would not play in vanilla

Only needed if you did not use the BSA Decompressor: Ogg Vorbis Libraries

Special install instructions:

  • Extract the .dll files from the archive to the root folder and overwrite when prompted

- Latest compiled Ogg Vorbis Libraries that allow the game to play certain .ogg files it wouldn't be able to play otherwise

New Vegas Reloaded - NVGE

Special install instructions:

  • Manually extract everything from the Data folder except the Music folder to the Fallout New Vegas Data folder
    • The .dll file can be ignored
  • Click the button on the top bar of MO2 and select INI Editor
  • Navigate to the FalloutCustom.ini tab
  • Add the following under the [Display] section
    • bFull Screen=0
    • Some users have reported this tweak needing to be made in the FalloutPrefs.ini tab

- Adds a new rendering pipeline, updates the vanilla shaders, and adds memory management

NVGE Custom .INI

Files to install:

For Users with an NVIDIA GPU Only: NVIDIA Profile Inspector

Special install instructions:

  • Extract the archive to a safe location
  • Run NVidiaProfileInspectorDmW.exe
  • Click Yes if there is a pop-up
  • From the Profiles: drop-down in the top left, select Fallout - New Vegas
  • In the 2. - Sync and Refresh section, change the following:
    • Frame Rate Limiter - 58 FPS
    • Maximum pre-rendered frames - 1
    • Vertical Sync - Force on
      • If you use G-Sync or Freesync, do not enable Vertical Sync
  • In the 5. - Common section, change the following:
    • Power management mode - Prefer maximum performance
  • Click the Apply changes button in the top right then exit

For Users with an AMD GPU Only: Tweaking the AMD Radeon Settings

  • Right-click on your desktop and select AMD Radeon Settings
  • In the Gaming tab, select the Fallout New Vegas profile
  • In the Profile Graphics tab, change the following:
    • Wait for Vertical Refresh - Enhanced Sync
      • If Enhanced Sync is not an option, select Always On
      • If you use Freesync, do not enable Wait for Vertical Refresh
    • Enable Chill (FPS)
    • Set Chill Min and Chill Max to 60
  • Hit the home button in the bottom left (the settings should automatically save), and make sure the word Enabled appears below the profile name
    • If not, click the three vertical dots to the right of the profile name and select Enable Profile
