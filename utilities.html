Essentials

We now begin the first section of New Vegas Rebuilt. This covers essentials - script-extending utilities many mods rely on alongside bug fixes and performance improvements.

I've temporarily removed recommendations for BSA Decompressor and Ultimate Edition BSM Fixes as I can't recall the exact steps I did to get them working with Vortex. They were definitely different to the Viva New Vegas ones. It was a simple process, but I think it's better to not include steps for them for now than risk messing up your New Vegas installation via incorrect instructions.

Collection 1a/10 (Utilities): Gopher's Stable New Vegas

This was the starting point for my modding journey. Gopher's Collection primarily includes various script extension tools many other mods rely on. Installing this Collection first will download and install most of the below 8 mods automatically.

4GB Patcher

You need to install this manually after downloading Gopher's collection! This is required for both Steam & GOG users! The 4GB Patch included with the GOG release is obsolete as it lacks NVSE integration. The other mods will be installed automatically through Vortex.

Installation instructions:

  1. Download the Main File - 4GB Patcher (you can skip this step if you downloaded Gopher's collection as it includes it)
  2. From the downloaded archive, extract the .exe file to the game's Root folder
  3. Right-click on FNVpatch.exe and select Run as administrator
  4. A command prompt window will open and should say FalloutNV.exe patched!
  5. Close the command prompt and a file named FalloutNV_backup.exe should appear in the game's Root folder
Makes the game large address aware, meaning it can use 4GB of RAM instead of 2GB.
From now on, you should run the game through the Play button on New Vegas in Vortex. This ensures the game runs as expected with your mods and xNVSE loaded.

Mod Limit Fix

Installation instructions:

  • Main File - MLF
Raises the game's file handles limit and improves performance/load times, even if you are far below it.

JIP LN NVSE Plugin

Installation instructions:

  • Main File - JIP LN NVSE Plugin
NVSE extension with new functions and many engine bug fixes.

JohnnyGuitar NVSE

Installation instructions:

  • Main File - JohnnyGuitar NVSE
Another NVSE extension similar to JIP LN.

New Vegas Tick Fix

Installation instructions:

  • Main File - NVTF
Works to fix micro-stuttering, improve performance, and allow playing at higher framerates.
In order for this mod to work properly, make sure you installed the DirectX and VC++ requirements from the Introduction page.

Yukichigai Unofficial Patch

Installation instructions:

  • Main File - YUP - Base Game and All DLC
Fixes thousands of bugs with both the base game and DLCs.

Unofficial Patch NVSE Plus

Installation instructions:

  • Main File - Unofficial Patch NVSE Plus
Fixes many bugs like YUP, but takes advantage of NVSE and JIP LN.

lStewieAl's Tweaks and Engine Fixes

Installation instructions:

  • Main File - Stewie Tweaks
  • Stewie Tweaks Essentials INI
Collection of many configurable quality of life improvements/bug fixes. Settings can be further enabled/disabled/tweaked in the new Tweaks section added to the main menu.

Utilities Testing

  1. Restart your PC (Otherwise some plugins will print out 0)
  2. Launch the game using the New Vegas option in MO2 (NOT the NVSE option)
  3. If you see a black console window appear, it means NVHR is working
    • If not, NVHR is installed incorrectly
  4. Once the game has reached the main menu, hit the ~ key (Above the tab key) to open the console
    • You will need to unplug/disconnect any controllers to be able to use the keyboard
  5. Enter GetNVSEVersion, and the console should print NVSE version: 6
    • If not, xNVSE is installed incorrectly
  6. Next, enter GetIsLAA, and the console should print GetIsLAA >> 2
    • If it prints >> 0, then the 4GB Patch was applied incorrectly
    • If it prints an error message, then JIP LN NVSE is installed incorrectly
  7. Next, enter GetINISetting "bUseThreadedAI:General" and the console should print INISetting bUseThreadedAI:General >> 1
    • If not, then the Tweaking FalloutCustom.ini step was done incorrectly
  8. Next, enter IsDLLLoaded CobbCrashLogger, and the console should print IsDLLLoaded "CobbCrashLogger.dll" >> 1
    • If not, then Crash Logger is installed incorrectly
  9. Next, enter IsDLLLoaded NVTF, and the console should print IsDLLLoaded "nvtf.dll" >> 1
    • If not, then NVTF is installed incorrectly
  10. Next, enter IsDLLLoaded mlf, and the console should print IsDLLLoaded "mlf.dll" >> 1
    • If not, then Mod Limit Fix is installed incorrectly

Collection 1b-c/10: Essentials

This is the first Collection for this guide. It adds many bug fixes, performance enhancers and further utilities designed to extend Gopher's Stable New Vegas. It also includes some HUD upscaling mods.

Aqua Performa Strip Performance Fix

Installation instructions:

  • Main File - Strip Performance Fix
Drastically improves performance on The Strip by getting rid of water LOD.

Clean Vanilla Hud

Installation instructions:

  • Main File - Clean Vanilla Hud
  • When running the installer only add 'Clean Map Icons' and check the "I am not a height indicator user"
Adds super clean, minimal, immersive minimap icons.

Combat Lag Fix (NVSE)

Installation instructions:

  • Main File - Fast Weapon Lag Fix
Improves framerate in combat by fixing an engine bug that lowered framerate while attacking an enemy with a health-bar visible.

Consistent Pip-Boy Icons

Installation instructions:

  • Download and install the first 2 Main Files normally (When prompted with a FOMOD installer, simply select Install, rename the 2nd file to match its file name)
  • Additionally install Clean Companion Wheel 256x256
Thousands of Pip-Boy icons re-made from scratch in HD along with many bug fixes.

Collision Meshes

Installation instructions:

  • Main File - Collision Meshes
  • Main File - 1.6.3 to 1.6.4 patch
replaces poor collision meshes allowing you to interact logically in ways you couldn’t before i.e. shoot through car windows, towers, railings etc..

Console Paste Support

Installation instructions:

  • Main File - Console Paste
Adds Ctrl+V as a hotkey to paste into the in-game console, along with a few other helpful shortcuts.

Crash Logger

Installation instructions:

  • Main File - Crash Logger
A DLL that logs raw crash data.

Elijah Voice Audio Files Fix

Installation instructions:

  • Main File - Elijah Voice Files Fix - WAV Encoding
Fixes audio glitches in Elijah's voice during Dead Money.

Extended Roombounds (exRB)

Installation instructions:

  • Main File - ExRB
  • Select 'full' in the installer and enable all experimental features
Changes roombounds around interior cells for a performance boost.

Extinguished Fire Barrels

Installation instructions:

  • Main File - Extinguished Fire Barrels
Replaces the fire effect in barrels with smoke for performance on lower-end PCs. Fire is one of the bigger FPS drop causes in the Gamebryo engine.

Fallout New Vegas Optimization Overhaul Reloaded

Installation instructions:

  • Main File - FNVOOR Base
  • Main OOR DLC
  • Main ESM
Optimises all loose Data files for a performance boost. However, the download is quite large at 3GB which is why this mod is optional.

FOV Slider

Installation instructions:

  • Main File - FOV Slider
Adds an FOV slider option to the MCM menu you'll set up in HUD. I'd recommend setting your FOV to 8o otherwise first person feels claustrophobic.

High Res Local Maps

Installation instructions:

  • Main File - High Res Local Maps
Increases the resolution of local maps to a configurable target resolution.

High Resolution Screens

Installation instructions:

  • Main File - High Resolution Screens
Increases the render resolution of in-game screens (Pipboy, terminals, character creation menu) to match game's screen resolution.

New Vegas Heap Replacer

Installation instructions:

  1. Download the Main File - NVHR using the Manual Download button
  2. From the downloaded archive, extract everything to the game's Root folder
Replaces the game's heap management system with a much faster version, which can lead to large performance gains.

New Vegas Mesh Improvement Mod

Installation instructions:

  • Main File - NVMIM
  • Main File - Plugin That Should Have Been In The Main File
  • Update File - Strip Collision Fix
Optimizations and fixes for over one thousand meshes throughout the base game and DLCs.

No Muzzle Flash Lights

Installation instructions:

  • Main File - No Muzzle Flash Lights
Disables the lights cast by muzzle flashes, which can greatly improve performance in combat.
This mod does not disable the actual muzzle flashes, but rather the lights the muzzle flashes cast on the environment. It also only disables the effect for NPCs by default, but that can be configured via the included INI file. See the mod's screenshots for details. Like any mod in the guide, it can be skipped if you like the effect and don't mind the performance hit.

Pip-Boy Shading Fix NVSE

Installation instructions:

  • Main File - Pip-Boy Shading Fix
Fixes Pip-Boy's shading, making it not ignore lights around it.

ShowOff NVSE

Installation instructions:

  • Main File - ShowOff NVSE
  • Main File - ShowOff INI
NVSE plugin adding new functions and engine-level tweaks & bugfixes.

Strip Lights Region Fix

Installation instructions:

  • Main File - Strip Lights Region Fix
Resolves the lights of the Strip not being visible from certain map regions.
You’ll have to install several compatibility patches for other mods in the other Collections throughout this guide. Make sure the Strip Lights Region Fix patch plugins are installed after Strip Lights Region Fix and its relevant mod in your Plugins tab.
