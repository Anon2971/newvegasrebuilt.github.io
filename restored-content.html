Restored Content

Now we get into the real modding. Over the 18 month development of New Vegas, content had to be cut due to performance, deadline or creative reasons. I’ve spent much time analysing TriangleCity’s excellent series to research that removed content in detail.

This section restores cut content I believe was cut solely due to deadline or performance issues so New Vegas more closely resembles what Obsidian Entertainment would have done with more development time. I haven’t included mods adding content I believe was removed for a reason or so far from completion it’s difficult to tell if it resembles anything Obsidian would have done.

Many mods are colored red as they add objects and/or NPCs. These are both known to impact performance. If you're playing New Vegas on an older PC, feel free to cherry pick which mods you'd like to install. Some NPC mods only enable them after completing certain quests and so they can be skipped if you think they're unnecessary.

Collection 3/10: Restored Content

This Collection contains all mods listed below.

Uncut Wasteland

Installation instructions:

  • Main File - Uncut Wasteland plus NPCs. Simple Open Freeside and Outside Bets compatible.
  • Optional File - Uncut Extra Collection. Simple Open Freeside and Outside Bets compatible.
  • Also install the relevant Strip Lights Region Fix patch
Restores many pieces of cut scenery and a few NPCs that were cut because of console limitations.

Prerelease Restored

Installation instructions:

  • Main File - Prerelease Restored v0.3.3
  • Additionally install the NavMesh Fix mod (Main Files - Prerelease Restored - NavMesh Fix)
Restores various miscellaneous objects throughout the Mojave Wasteland seen in prerelease footage likely cut for performance. Compliments and expands Uncut Wasteland.

Simple Open Freeside

Installation instructions:

  • Main File - Simple Open Freeside
Restores Freeside into a single large region as originally designed before being split into smaller areas due to console limitations. Modernised and optimised version of MoBurma's legendary original mod.

Ensure you install the below mod on top of this for bug fixes.

Simple Open Freeside: North Gate Fix

Installation instructions:

  • Main File - Simple Open Freeside - North Gate Fix
  • Optional File - Simple Open Freeside - Uncut Wasteland - VMS19 Script Patch
Fixes NPC pathing issues around the North Gate with Simple Open Freeside and Uncut Wasteland Extra Collection installed.

Dixon Thugs

Installation instructions:

  • Main File - DixonThugs - Simple Open Freeside
Restores 2 cut thugs placed near Dixon to Simple Open Freeside.

An in-game code comment fragment suggests this was cut by Obsidian early on as a design decision, but I think restoring it is a cool immersion booster anyway.

Freside Pickpockets

Installation instructions:

  • Main File - Freeside Pickpockets - Simple Open Freeside
Recreates an NPC programmed to calmly approach you, steal your caps and run away to Simple Open Freeside. Excellent immersion booster.

Simple Open Strip

Installation instructions:

  • Main File - Simple Open Strip
Restores The Strip into a single large region as originally designed before being split into smaller areas due to console limitations. Modernised and optimised version of MoBurma's legendary original mod.

The Strip NPCS Uncut - Content Resoration

Installation instructions:

  • Main File - The Strip NPCs Uncut
Restores an incredibly clever system that was left unfinished spawning 11 miscellaneous NPCs around you as you move down the Strip. They have unique behaviour sets and AI packages. Makes the Strip feel like a bustling, alive location as it was originally imagined to be. This was likely cut due to console performance concerns.

Simple Open Strip Zone Trigger Fix

Installation instructions:

  • Main File - Simple Open Strip Zone Trigger Fix Compatibility Edition
Fixes Strip NPCs Restored not functioning as expected with Simple Open Strip due to incorrect triggers in the middle region of The Strip. Hopefully at some point this fix will be implemented in Simple Open Strip itself.

Lucky 38 Lights Redone

Installation instructions:

  • Main File - Lucky 38 Lights Redone - Full Model
    • The Full Model version requires Strip Lights Region Fix v1.3 which you should have previously installed in Essentials.
Redoes the lighting around the Lucky 38 so it's bedazzling similarly to how it was originally envisioned.

Strip Enhancements

These mods combined are a mixture of cut content and minor visual enhacements. They make The Strip appear grander and closer to it's E3 2010 demo appearance.

Installation instructions:

Install the single Main File for each of these.
  • Ultra-Luxe Fountain Expanded
  • Cleaner Ultra-Luxe Fountain
  • Tops Casino Restored
  • Palm Trees Replacer
  • The Strip Planters Fixed

New Vegas Uncut - Panzermann's Edition

Installation instructions:

  • Main File - New Vegas Uncut
  • Main File - New Vegas Uncut - Outside Bets
An incredibly comprehensive cut content mod restoring miscellaneous quests, NPCs and alternative scenes such as being able to release The Thorn's animals, letting them slaughter all NPCs in that location. They'll also escape into Westside above and wreck havoc there. Remakes most of MoBurma's amazing New Vegas Uncut series with bug fixes and optimisations.

New Vegas Uncut 6 - If It Wasn't For Betsy Remastered

Installation instructions:

  • Main File - If It Wasn't For Betsy 2.5
Restores Betsy, a cut Brahmin Companion perfect for hauling your loot. They'll flee when combat occurs and there's also a surprise funny quest unique to them. They won't enter interiors due to their size. Remake of MoBurma's mod.

NPC Branch Expansion

Many NPCs were pared back for performance concerns including several at the climactic battle of New Vegas which had a much grander scope originally. Installing this selection of mods from Kvatchcount will restore several NPCs throughout the Wasteland. Most only appear after completing their related quests.

Installation instructions:

Install the single Main File for each of these.
  • Brotherhood of Steel Reinforcements
  • Brotherhood of Steel Robots
  • Followers Eyebot
  • Followers Help at the Dam
  • Great Khan Reinforcements

Legion Quests Expanded

Installation instructions:

  • Main File - Legion Quests Expanded - MAIN
  • Also install the relevant Strip Lights Region Fix patch
Due to the tight production schedule of New Vegas, the Legion faction scope was significantly reduced and several large areas dedicated to them had to be cut. This mod is the only one here that isn’t strictly cut content - instead it’s a quest mod that expands the Legion questlines in a way that IMO fits the vanilla vision.

A Cyber Affair - Mr. House Seduction Route Uncut

Installation instructions:

  • Main File - A Cyber Affair
Restores a cut seduction option for Mr. House if you're female and have the Black Widow perk. Also restores a cut female Securitron NPC that was removed due to VO recording issues.

Hoover Dam Jets Restored and Distant

Installation instructions:

  • Main File - Hoover Dam Jets Restored and Distant
  • Also install the relevant Strip Lights Region Fix patch
Restores a subtle but huge immersion booster of water jets doing their thing at Hoover Dam. Replaces the jets in Uncut Wasteland as these ones are also visible from a distance.

Great Kahns' Toughness Increased

Installation instructions:

  • Main File - Great Khans' Toughness Increased
One loading screen slide describes the Great Khans as fearsome warriors, but their base stats are the same as other factions. This mod increases the base toughness for Great Khans to better reflect that.

Honest Hearts - Tribal Race Restoration

Installation instructions:

  • Main File - Honest Hearts - Tribal Race Restoration
Restores custom races for Honest Hearts tribes that were cut for console performance.

Camp McCarran South Gate Restored

Installation instructions:

  • Main File - McCarran South Gate Restored
Restores a cut South Gate to Camp McCarran in a tiny (13kb!) package.

Mojave Outpost Car Apocalypse Uncut

Installation instructions:

  • Main File - Mojave Outpost Car Apocalypse Uncut
Restores many cut cars on the Mojave Outpost highway, but it may also impact performance if you’re running a more low-end PC. Be weary of gunfights as the cars can set off a massive explosion chain!
