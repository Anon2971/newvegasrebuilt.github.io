Restored Content

Now we get into the real modding. Over the 18 month development of New Vegas, content had to be cut due to performance, deadline or creative reasons. I’ve spent much time analysing TriangleCity’s excellent series to research that removed content in detail.

This section restores cut content I believe was cut solely due to deadline or performance issues so New Vegas more closely resembles what Obsidian Entertainment would have done with more development time. I haven’t included mods adding content I believe was removed for a reason or so far from completion it’s difficult to tell if it resembles anything Obsidian would have done.

Collection 3/10: Restored Content

This Collection contains all mods listed below.

Uncut Wasteland

Installation instructions:

  • Main File - Uncut Wasteland
A custom version of sandbox6's amazing Uncut Wasteland mod, which restores many pieces of cut scenery and a few NPCs that were cut because of console limitations.

Functional Post Game Ending Lite

Installation instructions:

  • Main File - Functional Post Game Ending Lite
Lets you continue playing after Hoover Dam without altering the game's world.

Essential DLC Enhancements Merged

Installation instructions:

  • Main File - Essential DLC Enhancements
Merge of many gameplay improvements for the DLCs, all updated and fixed.

Mojave Raiders

Installation instructions:

  • Main File - Mojave Raiders
  • Additionally install Mojave Raiders Tweaks
  • Additionally install JSawyer Ultimate - Mojave Raiders Patch and Mojave Raiders - Essential Vanilla Enhancements Merged Patch (select the name that matches the file on Nexus when the installation prompt comes up)
Major overhaul/balancing of the vanilla raiders, as well adding many new spawn points.

The Living Desert

Installation instructions:

  • Main File - The Living Desert - Main File 2.3
  • Additionally install The Living Desert - Essential Vanilla Enhancements Merged Patch (select the name that matches the file on Nexus when the installation prompt comes up)
Adds hundreds of new NPCs and scripted events that make the world feel much more alive and reactive.
