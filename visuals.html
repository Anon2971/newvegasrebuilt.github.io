Lighting

Collection 7/10: Lighting

This section impacts primarily the lighting and should have a relatively minimal impact on NV performance.

Desert Natural Weathers

Installation instructions:

  • Main File - Desert Natural Weathers NV
Major overhaul to all of the weathers for the base game and DLCs.
As visual mods are very subjective, not everyone will like DN Weathers. If you want an alternatives, I recommend checking out Altitude, Yellow Goodbye, or Clarity. Yellow Goodbye and Clarity aim for a much more vanilla style, mostly just removing the game's atmospheric tint. Meanwhile, Altitude aims to strike a balance between staying vanilla-friendly and being a complete overhaul. These three mods should have little to no performance loss, even on lower-end systems.

Atmospheric Lighting Tweaks

Installation instructions:

  • Main File - Atmospheric Lighting Tweaks - FNV
Makes changes to Image Spaces and Lighting Templates to give interiors a more dynamic and atmospheric look.

Climate Control NVSE

Installation instructions:

  • Main File - Climate Control NVSE
  • Optional File - Longer Weather Transitions
Fixes bugs and adds utilities for weather mods. The optional mod makes weather transitions take longer for realism.

Heat Haze

Installation instructions:

  • Main File - Heat Haze
Adds a distant heat haze visual effect during hot weather conditions. Uses Climate Control.

Cloud Shadows

Installation instructions:

  • Main File - Cloud Shadows
Changes Wasteland lighting when clouds are overhead. Great immersion tweak.

Dusty Distance Redone

Installation instructions:

  • Main File - Dusty Distance Redone
  • Optional File - Dusty Distance Redone - 39k (no nights tweaks)
Adds a dusty haze effect to the Wasteland only impacting your long-range view to make the world feel more expansive. I recommend the 39k version as I find the 29k makes the world feel small and it's more distracting than beneficial.

A Little More Lamplight

Installation instructions:

  • Main File - A Little More Lamplight
Enhances the vanilla streetlights and adds new ones around Outer Vegas, Freeside, and more.

Strip Street Light Corrections

Installation instructions:

  • Main File - Strip Street Light Corrections
Strip street light corrections.

MoonlightNVSE

Installation instructions:

  • Main File - MoonlightNVSE
Fixes moonlight by making the moon the light caster instead of the sun.

Night Sky Tweaks

Installation instructions:

  • Main File - Night Sky Tweaks
Improves night skylines for darkness and realism.

Windows of the Mojave

Installation instructions:

  • Main File - Windows of the Mojave
Adds interior windows where they don't match their exteriors.

Standalone Cateye and nightvision effects from RWL

Installation instructions:

  • Main File - Cateye Improved Main
Improved night vision visual effects.
