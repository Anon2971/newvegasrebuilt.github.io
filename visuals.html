Visuals: General Information

  • Home
  • General Information
  • Section Selection

Requirements:

  • A working installation of the Core module

Module Options:

  • The Visuals module has three options to choose from depending on your system specs; low, medium, and high:
  • Low - No major texture overhauls, just improved models/lighting. Very small performance impact if any
  • Medium - Texture overhauls for big objects, such as architecture, landscape, common clutter items, and weapons, but may leave a bit to be desired. Small to medium performance hit
  • High - Texture overhauls for most objects in the game. Medium to high performance hit and potentially a minor stability hit

Recommended Specs:

  • For the Low option, just about any system that can run the Core module can handle it
  • For the Medium option, any semi-modern PC with a quad-core (or dual-core with hyperthreading) that was decent single-core performance, 6+ GB or RAM, and a GPU with 3GB+ of VRAM can handle it
  • For the High option, a modern PC that can run new AAA games can handle it

My System Specs:

  • I personally run the Medium option, as it has the best performance/quality ratio and doesn't push the engine too hard. However, my system could run the High option
