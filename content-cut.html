Creating a New Profile:

In order to keep a full installation of other modules handy, we will create a copy of the profile:

  • Click the button on the top bar of MO2
  • Click on the profile of the most recent module you installed and select Copy
  • Input Viva New Vegas - Content for the new name then click OK
  • Select the Viva New Vegas - Content profile from the profile drop-down

Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Cut Content into the pop-up and select OK

Uncut Wasteland

Files to install:

  • Main File - Uncut Wasteland. Outside Bets
  • Optional File - Uncut Bison Steve Trees
  • Optional File - Uncut Cottonwood Cove Shack
  • Optional File - Uncut Greenhouses

FNVEdit editing required:

  • Run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to Uncut Cottonwood Cove Shack.esp and Interior Lighting Overhaul - Ultimate Edition.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Input 1392C5 into the FormID search box in the top left and hit enter
  • Right-click in the left pane and check Hide no conflict and empty rows
  • Drag and drop Cottonwood Cove Shack from Uncut Cottonwood Cove Shack.esp on to Cottonwood Cove Building 01 from Interior Lighting Overhaul - Ultimate Edition.esp
  • Save and exit

- Restores a huge amount of scenery and other random things that were patched out of the game post-release

Pre-Release Restored

Install normally


FNVEdit editing required:

  • Run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to PrereleaseRestored.esp and FNV Realstic Wasteland Lighting - All DLC.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Input 13C0B6 into the FormID search box in the top left and hit enter
  • Right-click in the left pane and check Hide no conflict and empty rows
  • Drag and drop the box to the right of the Object Bounds option from PrereleaseRestored.esp on to the same box from FNV Realstic Wasteland Lighting - All DLC.esp
  • Save and exit

- Restores content seen in the pre-release trailers and screenshots but later cut from the game
