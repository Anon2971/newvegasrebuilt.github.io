Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Mechanics into the pop-up and select OK

Enabling Fallout 3 Repair

  • Double-click on JIP LN in the left pane of MO2
  • In the INI-Files tab, click on NVSE/Plugins/jip_nvse.ini
  • Set bEnableFO3Repair to 1

- Enables the Fallout 3 repair system, where max repair health of items will be capped to the player's Repair skill.
  For example, if your repair skill is 75, you will only be able to repair items to 75% of their max health

Enabling More Features in lStewieAl's Tweaks

  • Double-click on lStewieAl's Tweaks INI (or the custom INI file if you installed that) in the left pane of MO2
  • In the INI-Files tab, click on nvse/plugins/nvse_stewie_tweaks.ini
  • Set bNoSkillTags to 1 (line 269, line numbers are only accurate when using the Custom INI file from the guide)
  • Set iRemoveTags to 2 (line 524)
    • Additional changes can be made under the [No Skill Tags] section at the bottom of the INI
    • Hides the [Speech], [Barter], etc. number tags from dialogue options, for example, instead of [Speech 75], it will say just [Speech]
  • Set bHideEnemyMarkers to 1 (line 50)
    • Removes enemy markers from the compass
  • Set bOverencumberedTweak to 1 (line 278)
    • Mimics the Fallout 76 encumbrance system
  • Set bEnteringVATSCostsAP to 1 (line 302)
    • Makes entering VATS key cost 10 AP
  • Set bChargedAttacksCostAP to 1 (line 308)
    • Makes charged melee attacks cost AP
  • Set bNoGrabOwnedItems to 1 (line 329)
    • Stops the player from grabbing items owned by NPCs
  • Set bWeightlessWornPowerArmor to 1 (line 392)
    • Removes the weight of the power armor when you are wearing it like in Fallout 4

BLEED

Files to install:

  • Optional File - BLEED NV

- Overhauls how damage and force is applied, making combat quicker and harder, and making previously unviable builds such as unarmed more viable

BLEED - No Always Knockdown Explosions

Files to install:

  • Main File - 3.2. Gameplay - BLEED No Always Knockdown Explosions

- Changes the "Always Knockdown" flag back to "Knockdown by Formula" for all explosions

Immersive Recoil 2.0

Install normally


Optional tweaking available:

  • Run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to ImmersiveRecoil.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Expand the ImmersiveRecoil.esp header in the left pane, and then the Globals header within it
  • These globals can be edited to enable/disable/change parts of the mod
    • See the mod description for details on what the globals do

- Adds configurable formula-based recoil to all weapons in the game

Super Mutant Overhaulord

Files to install:

  • Main File - Super Mutant Overhaul - No overlords

Manual FNVEdit editing required:

  • Run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to CIB Supermutants.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Input 15FFF0 into the FormID search box in the top left and hit enter
  • Right-click in the right pane and select Hide no conflict and empty rows
  • Right-click on the empty box above NightkinFaction in the CIB Supermutants.esp column and select Remove
    • This tweak removes the custom Nightkin faction that makes them aggro to the player from a Nightkin that is supposed to be passive
  • Save and exit

- Overhauls Super Mutants to be more of a threat to the player
