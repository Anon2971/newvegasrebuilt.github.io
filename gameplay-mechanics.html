Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Mechanics into the pop-up and select OK

Yukichigai Gameplay Tweaks

Install normally

- Compilation of many small gameplay tweaks to enhance gameplay while staying close to vanilla
  For a list of the compiled mods, see the included read-me
  This version is edited 99% by Lucas9 (a.k.a. Sigourn) for compatibility with many popular mods, with another 1% done by myself
  All credits go to Darthbdaman for the original mod, which was taken off of Nexus a while ago

Alternative Repairing

Install normally

- Adds specialized repair parts that can be crafted or bought on top of the vanilla repair system

Pickpocket Overhaul

Files to install:

  • Main File - Pickpocket Overhaul

- Overhauls the pickpocketing formula to take item weight and target's perception into account

Character Creation Plus

Install normally

- Modifies the items you get at the start of the game based on your tag skills and traits

Faction Armor Redone

Install normally

- Faction armor will no longer offset positive faction reputation, only negative

Loot All Vehicles

Files to install:

  • Main File - Loot All Vehicles DLCs

Hide the following:

  • Loot Vehicles.sp

- Loot all cars, trucks, golf carts, patrol cars, etc with opening/closing door sounds

Loot All Vehciles ESP Replacer

Files to install:

  • Main File - 3.2. Gameplay - Loot All Vehicles ESP Replacer

- Fixes ITMs, dirty edits, forwards YUP fixes, and balances the loot pool

Cannibal Reborn

Install normally

- Balances the Cannibal perk and adds immersive gore effects

Lonesome Road Flare Ammo

Hide the following:

  • Flare Ammo - ACS.esp

- Gives the Flare Gun from LR its own unique ammo type instead of using flamer fuel

Birds of a Feather Bear Arms Together

Files to install:

  • Main File - TLD_EnergyWeaponsForTheNCRNoStatChanges

- Integrates various energy weapons into the levelled lists and inventories of NCR troopers and rangers after completing the quest "Birds of a Feather"
