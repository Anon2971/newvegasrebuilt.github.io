Overhauls

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Overhauls.
This page contains mods that greatly overhaul the game by making it much more challenging (and more rewarding in turn) and adding new features. These mods are not recommended for a first playthrough of the game. It is especially important to thoroughly read the descriptions of these mods, so you know what has changed and how to use the new features. Every mod in this section should be considered to be subjective and as such they may not be suitable for everyone.

Mojave Arsenal

Installation instructions:

  • Main File - Mojave Arsenal
  • Optional Files - INI Config - Gun Runners' Arsenal Merged (select the name that matches the file on Nexus when the installation prompt comes up)
Adds ammo variants, reloading parts, and weapon mods as loot; provides comprehensive options for configuring and integrating Gun Runners' Arsenal; fixes item naming conventions; and improves recipes.

Player Combat Priority

Installation instructions:

  • Main File - Player Combat Priority
  • Optional/contextual - also install the The Bazar of Z - Strip NPCs Uncut - Don't Touch My Street Vendor patch so it functions as expected with Strip NPCs Uncut if you installed that
Adds various lively NPC behaviour AI packages for added immersion.

Mojave Raiders

Installation instructions:

  • Main File - Mojave Raiders
  • Additionally install JSawyer Ultimate - Mojave Raiders Patch and Mojave Raiders - Essential Vanilla Enhancements Merged Patch (select the name that matches the file on Nexus when the installation prompt comes up)
  • Additionally install Great Khans' Toughness Increased - Mojave Raiders Patch under Optional Files
Major overhaul/balancing of the vanilla raiders, as well adding many new spawn points.

The Living Desert

Installation instructions:

  • Main File - The Living Desert - Main File 2.3
  • Additionally install The Living Desert - Essential Vanilla Enhancements Merged Patch (select the name that matches the file on Nexus when the installation prompt comes up)
Adds hundreds of new NPCs and scripted events that make the world feel much more alive and reactive.
