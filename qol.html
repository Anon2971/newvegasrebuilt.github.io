Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input QOL Improvements into the pop-up and select OK

Collision Meshes

Files to install:

  • Main File - Collision Meshes FNV
  • Update Files - update Collision Meshes FNV 1.6.1 to 1.6.2

- Fixes collision meshes, the meshes which simulate where bullets, bodies, etc. can collide with an object, so you can now shoot through fences, car windows, etc.

Asterra's Many Fixes - Collision Meshes Patch

Files to install:

  • Optional File - Asterra's Many Fixes - Collision Meshes compatibility

- Patches the fixed meshes from Asterra's Many Fixes to also have the changes from Collision Meshes

Enhanced Camera

Install normally


Optional tweaking available:

  • Double-click on Enhanced Camera in the left pane of MO2
  • In the INI-Files tab, click on NVSE/Plugins/NVSE_EnhancedCamera.ini
  • All the features of the mod can be turned on/off and tweaked here
  • I personally change the following:
    • bFirstPersonDeath=0
    • bFirstPersonCannibal=0
    • bFirstPersonMisterSandman=0
    • fDlgFocusOverride can be set to 15.0 to remove dialogue zoom

- Enables a visible body and player shadows when in 1st person and changes 3rd person animations to 1st person

DiaMoveNVSE

Install normally


Special activation instructions:

  • Run the game and immediately exit
  • Double-click on the Overwrite folder in the left pane of MO2
  • Drag the NVSE folder into the DiaMoveNVSE mod in the left pane

Optional tweaking available:

  • Double-click on DiaMoveNVSE in the left pane of MO2
  • In the INI-Files tab, click on NVSE/Plugins/DiaMoveNVSE.ini
  • All the features of the mod can be turned on/off and tweaked here
    • I personally set fHeadTrackRangeMax to 0

- Enables diagonal movement for the player character

FOV Slider

Install normally

- Adds an MCM menu with options for adjusting all types of FOV

Motion Sickness Destroyer

Install normally

- Removes all motion blur, double vision, and other nasty nauseating effects that reduce visual clarity. Only install if those effects bother you

lStewieAl's Tweaks

Files to install:

  • Main File: Stewie Tweaks
  • Optional File: Stewie Tweaks INI

Mod activation instructions:

  • Run the game and immediately exit
  • Double-click on Stewie Tweaks INI in the left pane of MO2
  • In the INI-Files tab, click on nvse/plugins/nvse_stewie_tweaks.ini

A Note About Auto-Saves:

lStewieAl's Tweaks comes with a feature that replaces the vanilla auto-save system that will not corrupt saves like the vanilla system does. This tweak is enabled in the custom INI provided by the guide. Even if you do not use the preset, you should still enable the improved auto-save system to prevent save corruption.
