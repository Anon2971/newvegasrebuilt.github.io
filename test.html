LOOT

Special install instructions:

  • Run the installer and follow the wizard steps
  • On the last step of the installer, check the box adjacent to Launch LOOT then select Finish
  • Click OK to any pop-ups
  • Click the button in the top right to sort your load order
  • When it is finished, select Apply and then exit

- One-click load order sorter. Best used in combination with xEdit to view conflicts and LOOT's 'Metadata' feature to create custom sorting rules
