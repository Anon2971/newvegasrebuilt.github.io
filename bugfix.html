Bug Fixes

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Bug Fixes.

Yukichigai Unofficial Patch

Installation instructions:

  • Main File - YUP - Base Game and All DLC
Fixes thousands of bugs with both the base game and DLCs.

Unofficial Patch NVSE Plus

Installation instructions:

  • Main File - Unofficial Patch NVSE Plus
Fixes many bugs like YUP, but takes advantage of NVSE and JIP LN.

lStewieAl's Tweaks and Engine Fixes

Installation instructions:

  • Main File - Stewie Tweaks
  • Main File - Stewie Tweaks Menu-Config
  • Additionally install lStewieAl's Tweaks Custom INI (Install manually in MO2)
Collection of many configurable quality of life improvements/bug fixes. The INI file provided by the guide enables all of the bug fixes and many of the essential quality of life improvements. Settings can be further enabled/disabled/tweaked in the new Tweaks section added to the main menu.

ISControl Enabler

Installation instructions:

  • Main File - ISControl ESPless FULL
Dynamically adds ISControl node system animation support and easy per-weapon sights adjustment.

Vanilla Iron Sights Realligned

Installation instructions:

  • Main File - Vanilla Iron Sights Realligned
Fixes all the iron sights on vanilla weapon meshes using ISControl Enabler.

New Vegas Mesh Improvement Mod

Installation instructions:

  • Main File - NVMIM
Optimizations and fixes for over one thousand meshes throughout the base game and DLCs.

Improved Lighting Shaders

Installation instructions:

  • Main File - Improved Lighting Shaders
Fixes engine-level lighting bugs and allows for 4x the normal amount of active lights.

Climate Control NVSE

Installation instructions:

  • Main File - Climate Control NVSE
Fixes bugs and adds utilities for weather mods.

Elijah Voice Audio Files Fix

Installation instructions:

  • Main File - Elijah Voice Files Fix - WAV Encoding
Fixes audio glitches in Elijah's voice during Dead Money.

Pip-Boy Shading Fix NVSE

Installation instructions:

  • Main File - Pip-Boy Shading Fix
Fixes Pip-Boy's shading, making it not ignore lights around it.

Strip Performance Fix

Installation instructions:

  • Main File - Strip Performance Fix
Drastically improves performance on The Strip by getting rid of water LOD.
