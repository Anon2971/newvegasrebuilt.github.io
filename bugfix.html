A Note About Load Order and Conflict Resolution

LOOT is not used in this guide. The best way to sort your load order is to use FNVEdit to view the conflicts and manually sort your load order based off that and create a custom conflict resolution patch. The guide will provide a fully-modular handmade conflict resolution FOMOD for each module. This means that any mods listed can be left out, but if you plan on adding any big mods, familiarize yourself with manually solving conflicts in xEdit. I suggest this tutorial for familiarizing yourself with the tool and making custom conflict resolution. For more information about load orders and conflict resolution, see the FIXING CONFLICTS section here.

A Note About the Plugin Limit

Fallout New Vegas has a soft plugin cap of about 140 compared to the cap of 255 in Skyrim/Fallout 4. However, the stability of the game can decrease from around 110 onwards due to the limit varying on the size of plugins. The guide will use the tool zMerge (a.k.a zEdit) to merge plugins together to stay below the limit. If you plan on adding your own mods in addition to the ones in the guide, familiarize yourself with the tool so you can stay below the limit as well.

A Note About MenuMode 4 Scripts

MenuMode 4 is a method of activating a script added by JIP LN that fires the script when the main menu loads. In order to give scripts that fire in MenuMode 4 a chance to run, you must wait on the main menu for a second or two before starting the game

Unofficial Patch Plus

Files to install:

  • Main File: Unofficial Patch Plus
  • Optional File: Unofficial Patch Plus - Addendum

Optional FNVEdit editing recommended:

  • Once the mod is installed, run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to Unofficial Patch Plus.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Input 15C570 into the FormID search box in the top left and hit enter
  • In the right pane, change the Magnitude of the record below ReduceCarryWeight in the Unofficial Patch Plus.esp column from 75 to 0
    • If you do not see the ReduceCarryWeight record, right-click in the right pane and un-check Hide no conflict rows
      • This tweak removes the penalty to companions' carry weight, as I know a lot of people wouldn't like that in a bug fix mod
  • When you exit FNVEdit, you should be prompted to save the edited plug-in
  • Make sure Unofficial Patch Plus.esp is checked and Backup plugins is unchecked then click OK

- Collection of supplementary bug fixes to be used alongside YUP
NOTE: This mod contains what some might call "gameplay changes." I personally look at them as continuity/QOL fixes, but if you don't want these "gameplay changes," you can either not install the optional file which contains these changes, or try Unoffical Patch NVSE as an alternative

Weapon Mesh Improvement Mod

Install normally


Manual FNVEdit editing required:

  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to WMIMNV.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Input 05000906 into the FormID search box in the top left and hit enter
  • Right-click in the right pane and select Hide no conflict and empty rows
  • Drag and drop the value next to Tracer Chance from FalloutNV.esm on to the value from WMIMNV.esp
    • Fixes a bug that caused Dragon's Breath rounds to cause micro stutter when fired
  • Input 3a15e into the FormID search box in the top left and hit enter
  • Drag and drop WeapNVLightMachineGun "Light Machine Gun" [WEAP:000906DF] from FalloutNV.esm on to the blank box directly next to it from WMIMNV.esp
    • Adds LMGs back to the FormID list of weapons that Super Mutants can use to fix them spawning with them but not being able to use them
  • Input 121154 into the FormID search box in the top left and hit enter
  • Right-click on WMIMNV.esp on the top bar of the right pane and select Remove
  • Save and exit

- Fixes mesh errors, UV errors, incorrect flags, missing extra data, form lists, projectiles, and other weapon related bugs
NOTE: Due to the nature of this mod, any other mods that edit the same weapon meshes will be incompatible. This is why you won't see the likes of mods such as WMX or EVE.
