Gameplay+

Collection 5/10: Gameplay+

These mods extend the core design and gunplay. Individually they don't stray too far from the original gameplay, but combined together the gameplay considerably in several categories. Skip this section entirely if you want to be a vanilla gameplay purist.

Delay DLC Redux

Installation instructions:

  • Main File - Delay DLC Redux
Prevents the game's DLC from spamming the new game startup messages and requires you be near a given DLC's starting location to receive its quest.

Better Character Creation

Installation instructions:

  • Main File - Better Character Creation
Improves the character creation by speeding up the process, adding specialized gear based on your tag skills, and making Wild Wasteland an opt-in feature rather than a trait. Recommended for second playthroughs onwards.

Improved Quest Objectives

Installation instructions:

  • Main File - IQO 2.5
  • In the installer enable all DLC alongside Legion Quests, YUP, Uncut and Living Desert patches (the latter will be installed later on)
Rewrites all the quest objectives for clarity, better grammar and more consistent labelling. This will make quests easier to follow.

Too Many Quest Markers

Installation instructions:

  • Main File - Too Many Quest Markers
  • Main File - Too Many Quest Markers - Honest Hearts
  • Main File - Too Many Quest Markers - Dead Money
  • Main File - Too Many Quest Markers - Old World Blues
  • Main File - Too Many Quest Markers - Lonesome Road
Removes quest markers for moments where you’re required to search for your next objective, then adds the marker to the map once you’ve found it. This adds massively to immersion since it emphasises world exploration i.e. when someone says you have to find someone, you have to actually find them via paying attention to the NPC directions rather than blindly follow an objective marker. New Vegas actually gives you a lot of directions through its dialogue, so this is an excellent immersion booster to make you feel like a real Mojave desert inhabitant.

Improved Quest Marker Descriptions

Installation instructions:

  • Main File - Quest Improvements 1.01
Extends the above by improving quest objectiives when NPCs very occasionally don’t make it clear where to go.

Combat Overhaul

The below mods combined dramatically overhaul and expand the combat mechanics.

Mojave Arsenal

Installation instructions:

  • Main File - Mojave Arsenal
  • Optional Files - INI Config - Gun Runners' Arsenal Merged (select the name that matches the file on Nexus when the installation prompt comes up)
Adds ammo variants, reloading parts, and weapon mods as loot; provides comprehensive options for configuring and integrating Gun Runners' Arsenal; fixes item naming conventions; and improves recipes.

Just Assorted Mods

Installation instructions:

  • Main File - Just Assorted Mods
  • Additionally install Clean Just Weapon Wheel
  • Open the kNVSE ini file by going to the Mod, right clicking→Open in file manager and ensure bCheckForSpecialIdle=0 so sprint customisation functions properly
Highly configurable set of new modern features such as a quick loot menu, dynamic crosshair, sprinting, and more. Settings can be tweaked using the Mod Configuration Menu found in the in-game pause menu. I recommend enabling only Just Sprint with the speed boost set to 60% for your first playthrough so you can appreciate how the original New Vegas combat felt, then enable the other features as you wish for your second playthrough onwards.

Just Sprint Animation Replacers

Installation instructions:

  • Main File - New sprint animations for JAM
Much better animations when sprinting.

NPCs Sprint

Installation instructions:

  • Main File - NPCs Sprint In Combat
Allows enemies to use the new sprint feature from JAM.

Empty or Not Empty?

Installation instructions:

  • Main File - Empty or Not Empty
  • In the Mod files (ight click→Open in file manager), open config/Empty or Not Empty.ini and set bNVAFNAC=1 and bNVMIM=1 for Animated Containers and Mesh Improvement Mod si[[prted
Prevents the player from magically knowing whether or not a container or a corpse is empty before having checked it. This is redundant if you use the JAM Loot Menu since you will be able to instantly check all containers with it enabled.

Player Combat Priority

Installation instructions:

  • Main File - Player Combat Priority
Makes enemies focus on the player in combat rather than solely companions.

Simple Fast VATS

Installation instructions:

  • Main File - Simple Fast VATS
  • Also enable the faster VATS option in Tweaks as you wish
VATS opens faster.

FPS Leaning

Installation instructions:

  • Main File - FPS Leaning
  • Set the default shortcuts to Q/E in the Mod Configuration Menu as this accompanies the default WASD movement controls perfectly
  • Change any clashing default controls in the Controls menu - for instance I set 'Use' to F and 'Auto-Walk' to /
Adds modern leaning mechanics to combat when aiming down sites in first person.

See Through Scope Pack

Installation instructions:

  • Main File - See Through Scope Pack(Dedicated Animation)
  • Also install the See Through Scope Pack Vanilla Option Fixes mod to resolve some bugged weapons
Replaces the outdated full-screen weapon scopes to modern, realistic POV ones.

The Dedicated Weapons variation was chosen as the Anti-Material Rifle won't have the correct scope with the Hit anims variation. This is why I recommended the Dedicated Animations anim replacement pack earlier instead of Hitman's well-known ones.

Weapon DOF and Blur Effects

Installation instructions:

  • Main File - Weapon DOF and Blur Effects
Adds a subtle weapon blur when aiming down sites.

Cover Based Stealth Mechanics

Installation instructions:

  • Main File - Cover Based Stealth v1-09
  • When you load up New Vegas after installing this mod, you will see pop-ups providing config options. Select 'Vanilla FNV' and 'Basic Cover Stealth'. Use the MCM to choose different options as you'd like to
Rescripts the stealth mechanics for difficulty and realism with updated enemy cone of vision etc.

Dynamic Sealth -- Lighting Modifies Sneak

Installation instructions:

  • Main File - CoverOfDarkness v3.0 (Dynamic)
  • Enable 'More Frequent Light Updates' in Tweaks so the Sneak Skill modifier updates more dynamically
Adds a Sneak Skill modifier (-5 to +20) depending on how much light your character is under. Stealth mechanic immersion booster that makes logical, real world sense IMO.

Progressive Sneak Attack Damage

Installation instructions:

  • Main File - Progressive Sneak Attack Damage
Adjusts the Sneak damage multiplier so that it increases as you increase your Sneak skill rather than always be 2x. That damage mutiplier is now only enabled at a Sneak skill of 75 or higher. Difficulty rebalance since Sneak being 2x damage right from the start makes the stealth route in New Vegas way overpowered.

Dymamic Combat Styles

Installation instructions:

  • Main File - NPC Dynamic Combat Styles
Adds dyanmic fighting strategies to NPCs that can change mid-combat.

Martialize

Installation instructions:

  • Main File - Martialize
Adds melee attack to armed NPCs you when you get too close.

Awesome Staggering

Installation instructions:

  • Main File - Awesome Staggering
Replaces default stagger animations with more lively ones of a comparable length. Adds to immersion.

BURN - Hardcore Fire Effects

Installation instructions:

  • Main File - BURN - Hardcore Fire Effects
Completely reworks fire mechanics for realism and fun. Now fire is as dangerous as it should be. Enemies will panic when set on fire as will your companions (this mod restores some very intense and scary cut 'on fire' voice-overs), but the same goes for yourself. If you get set on fire yourself expect to die very quickly! Watch out for fire-wielding Raiders!

Helmet Armor Rebalance (JIP)

Installation instructions:

  • Main File - Helmet Armor Rebalance (JIP)
Changes how helmet armor is handled for improved, increased difficulty.

Perks Overhaul

This curated selection of perk mods updates and upgrades individual perks and perk-related mecahnics in vanilla-friendly form.

Installation instructions:

Install the Main File for each of these mods.
  • Simple Perk Buffs
  • Well Rested Overhaul
  • Light Step for Followers (Flyer Variation) - get the Flyer Variation
  • Immersive Four Eyes Visuals - adds a blur if you're not wearing glasses and have the Four Eyes perk

Gameplay Overhaul

The below mods combined dramatically overhaul and expand non-combat gameplay mechanics.

Miscellaneous Tweaks Collection

Installation instructions:

  • Main File - Miscellaneous Tweaks Collection
Several useful gameplay improvements combined like additional fast travel markers in The Strip and Freeside, enemies that can shoot from further away, Gun Runner Arsenal ammo is only available from high-end vendors, higher Strip entry cost (2k to 5k caps) encouraging exploration etc.

Skillful Lockpickery (or Skyrim-Style Lockpicking)

Installation instructions:

  • Main File - Skillful Lockpickery - A Lockpick Reworkery
  • Enable 'Pick Locks Without Skill' within Stewie Tweaks so you can try any lock like Skyrim
Reworks the lockpicking system so the difficulty of each lock is based on your own skills versus the lock difficulty. Makes the lockpicking mildly difficult instead of a cakewalk. Be careful with your Bobby Pins.

Pointlight Flashlight (and Piplight)

Installation instructions:

  • Main File - Pointlight Flashlight
Adds flashlights activated by holding B. You can find it amongst clutter in containers or craft one with 20 Repair skill, 2 Energy Cells, 1 Shot Glass, 1 Scrap Electronics, and 1 Leather Belt. Use this as a more atmospheric Pipboy light for exploring creepy locations like dungeons or when travelling at night.

Purchase Items on Display (xNVSE)

Installation instructions:

  • Main File - Purchase Items on Display
Adds the ability to purchase items being displayed in shops instead of only being able to steal them.

Sometimes the 'Purchase' text can still appear red as if you're stealing the item. This can be ignored. The author hasn't figured out how to fix this yet.

Repair Condition Matters

Installation instructions:

  • Main File - Repair Condition Matters - Chocolate ESPless
Changes how Repair works so that the condition of the weapon you picked up influences how much it can repair more than your Repair skill. Makes repairing weapons take many more weapon pickups from NPCs for balance.

Simple Helmet Effect Toggle

Installation instructions:

  • Main File - Simple Helmet Effect Toggle
  • When using the installer go for 'Vanilla Plus' if planning to play without the Graphics Collections; otherwise install the 'ADAM Reborn' variant
Press N to toggle night vision when wearing one of the handful of armors with night vision (NCR Ranger, Riot Gear or Power Armor).

Westside Reputation

Installation instructions:

  • Main File - Westside Reputation
Adds reputation to a town containing several quests that should logically have it.

New Vegas Karma Rebalanced

Installation instructions:

  • Main File - New Vegas Karma Rebalanced
  • If you decide to install JSawyer Ultimate Edition, also install the Miscellaneous File - JSUE Patch
Rebalances karma values so your actions (dialogue/NPC killing) have a more noticable impact on your Good/Evil alignment.

It's a Sin to Tell a Lie

Installation instructions:

  • Main File - It's a Sin to Tell a Lie
Fun immersion tweak adding minor negative Karma when you lie in dialogue.

Radiation Overhaul

The below mods combined significantly expand the radiation mechanic so its something you have to actively manage rather than be a minor, somewhat pointless annoyance you can easily ignore.

RAD - Radiation (is) Actually Dangerous - Overhaul

Installation instructions:

  • Main File - RAD - Overhaul
  • If you choose to install JSawyer Ultimate Edition, turn on JSawyer Increments in the MCM Menu for Hardcore Mode compatibility
An excellent radiation overhauler that causes RADs to reduce your maximum health similarly to Fallout 4. Shown via red bars blocking off the relevant portions of your HP. Also impacts Action Points in Hardcore Mode.

Radiative Muck

Installation instructions:

  • Main File - Radiative Muck
Makes pools of muck scattered across the Wasteland give off radiation as they logically should.

Irradiant Muck

Installation instructions:

  • Main File - Less Irradiant Muck
Replaces the radioactive muck pool texture with a sickly green one. Immersion booster and acts as a visual warning for the now-irradiated muck pools.

Ghouls Inflict Radiation Damage

Installation instructions:

  • Main File - Ghouls Inflict Radiation Damage NV
Adds logical radiation damage to Ghouls.

Even More Accurate Giger Clicking

Installation instructions:

  • Main File - Even More Accurate Geiger Clicking (without rad meter)
Increases the Giger counter sound frequency to be more realistic. Bumps up the creep factor when exploring radioactive locations. Also makes the Atomic! perk effects kick in right away rather than after one second of radiation.

Misc. Gamesetting Tweaks

Gamesetting tweak presets with an easily confiurable installer. I recommend enabling the following:

The Big Gameplay One

The below mod is what I consider the most essential gameplay overhaul mod. I'd recommend saving it for your second playthrough onwards as it dramatically changes the default difficulty balance of New Vegas to something with much more emphasis on role-playing and survival.

JSawyer Ultimate Edition

Installation instructions:

  • Main File - JSawyer Ultimate Edition
  • Optional File - JSawyer Ultimate Edition - Push's Tweaks (select the name that matches the file on Nexus when the installation prompt comes up)
  • Additionally install JSUE - YUP Patch (select the name that matches the file on Nexus when the installation prompt comes up)
An updated and fixed version of the gameplay re-balance mod made by the lead director of the game, Josh Sawyer. It sigifcantly reworks item weight, weapon damage and perks to encourage you to carefully choose the type of character you want to play.
The INI for this mod contains many configurable features, such as an adjustable level cap and carry capacity formula. You should look through the INI and see if there are any features you would like to change. For example, you can set fCarryWeightBase to 150 if you want the vanilla carry weight formula back.
If you followed the Restored Content section and installed the mods recommended there, you must reinstall DiDisaan's Patch Emporium after installing JSawyer Ultimate Edition to make sure a compatibility patch for Legion Quests Expanded is added. You must also reinstall Brave New World to add a compatibility patch if you installed it as part of following Gameplay HD.

Optional Difficulty Enhancers

The below mods are optional. Combined together they will make your New Vegas experience much more difficult and make you feel like a real surviver of a post-apocalyptic wasteland. I personally play with all these mods enabled as well.

Famine

Installation instructions:

  • Main File - Famine
Dynamically lowers the amount of loot in the world to create a harsher and less forgiving economy.

Food Effect Tweaks

Installation instructions:

  • Main File - Food Effect Tweaks
  • If you installed Misc. Gamesetting Tweaks, disable 'Twice the Rads' as Food Effect Tweaks already rebalances how much radiation you get from consumables
Lowers the amount of healing provided by food items, making the distinction between the Survival and Medicine skills more important. Also reworks the radiation amounts for consumable items.

Economy Overhaul

Installation instructions:

  • Main File - Economy Overhaul
Rejigs the default value of many loot items and how many caps vendors have by default forcing you to more carefully scavenge for caps.

Water Overhaul

Installation instructions:

  • Main File - Economy Overhaul
  • also install the JSUE Water Overhaul Patch
Creates a new universal bottling system allowing you to gather water from water sources by pressing Z. Drinking any liquids will grant you a new Empty Bottle item allowing you to fill them from any water source. Water sources have varying levels of purity (radiation) to ensure this is balanced and you can additionally purchase water from vnedors.

BLEEDLESS

Installation instructions:

  • Main File - BLEEDLESSNV
  • Optional File - Better Damage Multipliers
  • Optional File - Ragdolls Patch
Significantly increases combat difficulty via limb-specific multipliers and deadly explosives for tough, realistic combat. It works nicely with Famine’s limited loot drops as the limited ammo means every bullet counts. You can one-hit enemies, but they can one-hit you. Think STALKER. Save often and be ready to strategise non-companion/solo battles.

Progressive Difficulty

Installation instructions:

  • Main File - Progressive Difficulty
Even with all the above mods installed, as you progress through New Vegas your gear significantly improves to the point that most combat becomes relatively easy. The opening ten hours are tough, but beyond that point the difficulty gets average. This mod scales damage slightly as you level up in the favour of your enemies to somewhat rectify that.

Optional Charisma Enhancers

The below mods combined make the Charisma stat more important and useful. In my opinion it was somewhat a design decision to make Charisma useless for the sake of fun, so these are optional. I personally play with all 3 installed.

Installation instructions:

Install the Main File for these mods.
  • Follower Formula Redone - your Charisma divided by 2 determines how many Companions you can have whilst adding a "Magnetic Personality" perk to increase your Companion limit by 1
  • Actually Useful Charisma - makes some perks require 6 Charisma and changes several dialogue Speech checks to use Charisma instead when it makes logical sense
  • More Charisma Checks - converts more Speech checks to Charisma checks when it makes logical sense

Optional Gameplay Enhancers

The below mods add some additional gameplay features. They're optional as they fun additions rather than anything that dramatically changes New Vegas. I personally play with only Immersive Sleeping Encounters and Better Hacking enabled. Sometimes Hardcore Hacking too.

Immersive Sleeping Encounters

Installation instructions:

  • Main File - Immersive Sleeping Encounters
Adds a 30% chance of random DnD-like events occuring via a dialogue box after sleeping such as loot going missing. Roleplay immersion booster.

Immersive Fast Travel Encounters

Installation instructions:

  • Main File - Immersive Fast Travel Encounters
Adds a 30% chance of random DnD-like events occuring via a dialogue box after fast travel. Roleplay immersion booster.

Better Hacking

Installation instructions:

  • Main File - Better Hacking
Adds a Password Bypass item in containers that have Bobby Pins allowing you to skip Terminals AKA the hacking minigame if you meet its minimum Science Requirement. By default using the item is programmed to E. Adjust the .ini file to change the default key as you wish.

Hardcore Hacking

Installation instructions:

  • Main File - Hardcore Hacking
Replaces the hacking minigame with a new hangman system. Guess the word based on a few letters provided. More letters are given as your Science level increases. Very difficult replacement for the default minigame but also a lot more genuinely challenging.

Leave No Witnesses

Installation instructions:

  • Main File - Leave No Witnesses
Killing members from a faction reduces your reputation for them. Unless… Nobody was there to see who killed them? This mod is a very cool immersion booster as killing members of a faction won’t change your rep if there’s no witnesses. However, it somewhat breaks the original design forcing you to pick a side. Install at your discretion.

Caravan Overhaul Compilation

Installation instructions:

  • Main File - Hardcore Hacking
Tucked away among some New Vegas NPCs is a fantastic card game. If you’ve not played card games before it can feel like a pointless distraction, but if you digest the rules and give it a few plays it turns into an excellent bit of (cap-rewarding!) fun when you’re bored of shooting. This mod beefs up the AI decks and adds caravan players. This makes Caravan pretty damn hard, so install it at your discretion.
