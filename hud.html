User Interface

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator User Interface

User Interface Organizer

Installation instructions:

  • Main File - UIO - User Interface Organizer

- Automatically ensures that multiple HUD extensions work together

The Mod Configuration Menu

Installation instructions:

  • Main File - The Mod Configuration Menu
  • Update File - MCM BugFix 2

- Allows select mods to be configured via a single in-game menu

DarnUI

Installation instructions:

  1. Download the mod and install manually through MO2
  2. Additionally install DarnUI Config (Install manually in MO2)

- Greatly improves the look and scale of the UI, especially on larger monitors

If you do not like the look of DarnUI, you can instead try Vanilla UI Plus. Installation instructions can be found in the FAQ.

Contextual HUD

Installation instructions:

  • Main File - Contextual HUD
  • Optional File - Contextual HUD - DarnUI Patch

- Makes HUD elements only show under certain conditions, configurable via an INI

Consistent Pip-Boy Icons

Installation instructions:

  1. Download and install the first 2 Main Files normally (if prompted with a FOMOD installer, simply select Install)
  2. Additionally install Clean Companion Wheel

- 1000s of Pip-Boy icons re-made from scratch in HD along with many bug fixes

High Res Local Maps

Installation instructions:

  • Main File - High Res Local Maps

- Increases the resolution of local maps from 128 to 512 and optionally 1024

High Resolution Screens

Installation instructions:

  • Main File - High Resolution Screens

- Increases the render resolution of in-game screens (Pipboy, terminals, character creation menu) to match game's screen resolution
