User Interface

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator User Interface

User Interface Organizer

Installation instructions:

  • Main File - UIO - User Interface Organizer

- Automatically ensures that multiple HUD extensions work together

The Mod Configuration Menu

Installation instructions:

  • Main File - The Mod Configuration Menu
  • Update File - MCM BugFix 2

- Allows select mods to be configured via a single in-game menu

HUD Only Explosion Shake

Installation instructions:

  • Main File - HUD Only Explosion Shake

- Changes camera shake to only shake the HUD rather than the entire screen, creating a much less nauseating effect while still keeping the addded atmosphere

DarnUI

Installation instructions:

  1. Download the mod and begin the installation normally through MO2
  2. Additionally install DarnUI Config (Install manually in MO2)

- Improves UI without being comically big

Vanilla HUD Cleaned

Installation instructions:

  1. Download the main file and begin the installation normally through MO2
  2. You will be prompted with a FOMOD installer, in which you should deselect Clean Fonts, Vanilla UI+ Patch and I am not a Height Indicator User, then enable Clean Fonts for DarnUI and select Install
  3. Additionally install Clean Companion Wheel

- Hand-made up-scaled/cleaned recreations of many vanilla HUD elements

Consistent Pip-Boy Icons

Installation instructions:

  1. Download and install all three Main Files normally (if prompted with a FOMOD installer, simply select Install)
  2. Additionally install Pop-Up Message Icons (You will be prompted with a FOMOD installer, select Extension - Hunger, Dehydration, Sleep and Radiation and then Install)

- 1000s of Pip-Boy icons re-made from scratch in HD along with many bug fixes
