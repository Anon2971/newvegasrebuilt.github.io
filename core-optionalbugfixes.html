These are small and simple mods that can easily be added on top of any mod setup that don't require any conflict resolution and can be merged

Enslaving Arcade Fix

Install normally

- Allows you to give Arcade to Caesar at any point during the 'Et Tumor, Brute?' quest

Female White Glove Society Mask

Install normally

- Adds a separate female mesh for the White Glove Society Mask to fix alignment issues

Melee Reach Fixed

Files to install normally:

  • Optional File - ALTERNATIVE mastahbossu's Scripted Version

- Fixes the range of melee weapons to better represent their size

Sierra Madre Exterior Fixed

Install normally

- Fixes the messed up texture on the Sierra Madre exterior

Psycho and Slasher Bonus Damage Fix

Install normally

- Fixes the Psycho and Slasher damage buffs not stacking

Clicking Ambient Sound Fix

Install normally

- Fixes faint clicking in some ambient tracks that most people won't hear, but some audiophiles might

No Muzzle Flash Lights

Install normally

- Removes the lights casted on the environment by muzzle flashes, which can be very performance intensive especially on weaker systems

No Snow in Globes

Files to install normally:

  • Main File - No Snow In Globes
  • Optional File - No Snow for Honest Hearts

- Removes the snow effect in snow globes which can massively boost FPS in the Lucky 38, even on high-end PCs

Motion Sickness Destroyer

Install normally

- Removes all motion blur, double vision, and other nasty nauseating effects that reduce visual clarity. Only install if those effects bother you

Faster Pip-Boy Animation

Files to install normally:

  • Main File - Faster Pip-Boy Animation (2x)

- Doubles the speed of the Pip-Boy opening animation
