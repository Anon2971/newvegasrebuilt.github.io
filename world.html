Collection 9/10: World Overhaul

The below mods overhaul locations scattered throughout the Mojave Desert in vanilla-friendly form. They're primarily visual overhauls.

New Vegas Landscape Overhaul Re-Remastered

Installation instructions:

  • Main File - New Vegas Landscape Overhaul Re-Remastered
Adds many extra objects and details to the quieter parts of the Mojave Wasteland. Also restores cut buildings to Primm and and Westside. LOD should be regenerated after installing this since it adds a lot of large objects.

Enhanced Landscapes

Installation instructions:

  • Optional Files - Enhanced Landscapes for NVLO
Similar concept to the above covering more locations.

NV Interiors

Installation instructions:

  • Main File - NVInteriors Core v2.1.1
  • Main File - _NVInteriors_Combo Edition v1.3.4
  • Download the Main File from A Room with a View for a better Tops and Gammorah Suite interiors
  • Download NVInteriors Rewritten - Main Files - NVInteriors_Rewritten only
  • Download NVInteriors Remastered
Adds numerous new building interiors to explore that are so well designed I genuinely couldn’t tell what was and wasn’t in the original game. Rewritten and Remastered improve the writing and items for a more stable, less save bloat, vanilla-like experience.

Better Bases - NCR

Installation instructions:

  • Main File - Better Bases - NCR
Large-scale overhaul of all NCR faction locations in the base game. Fixes floating decals and items and enhances level of detail and immersion through better world building.

Better Bases - Kings

Installation instructions:

  • Main File - Better Bases - Kings
Overhaul of The King's School of Impersonation (interior only). Fixes floating decals and items and enhances level of detail and immersion through better world building. Adds especially useful direction arrows for finding your way out on the upper floors as it has a somewhat confusing layout.

Camp McCarran Animated Monorail

Installation instructions:

  • Main File - Camp McCarran monorail animation stand alone version
  • Update File - Monorail Proper Sounds
Adds an animated monorail car to Camp Mccarran’s exterior visible all across the Wasteland as you traverse across the plains. Subtle but huge immersion booster making the base feel like a living location.

Vikki & Vance Improvements

Installation instructions:

  • Main File - Vikki & Vance Improvements
Primm is pretty beat up when you first stumble across it. This mod makes it so its casino has a pleasant boost of life once its key quests have been finished.

This mod will break Primm if installed in the middle of a save, so please only install it before starting a fresh save!

Lucky 38 Presidential Suite Strip View Windows Tweaked Open Strip

Installation instructions:

  • Main File - Strip Open Version
Overhauls the Lucky 38 Suite for visual spleandour. It makes your suite feel special rather than a random room in an old casino. It might be too bright for some tastes compared to the lonely original vibe, but it’s really well done IMO. This is a tweaked version with minor improvements i.e. an open strip view through the windows vs vanilla strip view.

The Final Resting Place

Installation instructions:

  • Main File - The Final Resting Place - A Lucky 38 Control Room Overhaul - Outside Bets
The concept art for one important location was much grander than the final implementation. This excellent mod expands it into something closer to its original concept art.

Metro Inspired Sewers

Installation instructions:

  • Main File - Metro Inspired Sewers
Reworked sewers with clutter and other details so they feel more lived in.

Believable Grub n Gulp

Installation instructions:

  • Main File - Grub n Gulp but better
Reworked Grub n Gulp location.

Lux Aeterna

Installation instructions:

  • Main File - Luxe Aeterna - The True Revival of Luxury
Gorgeous, vanilla-friendly Ultra-Luxe overhaul.

Hidden Valley Overhauled

Installation instructions:

  • Main File - Hidden Valley Overhauled Classic BoS Theme
Prettier but vanilla-friendly Hidden Valley overhaul. Also replaces the default music with the Brothood of Steel theme since it makes sense lore wise.

Vault 22 Flora Overhaul Remastered

Installation instructions:

  • Main File - Vault 22 Flora Overhaul Remastered
  • Also install the Strip Lights Region Fix patch
  • Also install Vault 22 Animated Foilage for animated Vault 22 plants
Breathes new life and atmosphere into Vault 22. A complete remaster of Macintroll's Vault 22 Flora Overhaul.

Casino Crowds

Installation instructions:

  • Main File - Casino Crowds
The New Vegas casinos were shown filled with NPCs early on in development before being parred back due to performance issues. This mod doubles the number of NPCs in all the Strip casinos to make them feel more livlely as they should be. This may cause performance problems on low-spec PCs, so install with caution.

Lime's Fort Overhaul

Installation instructions:

  • Main File - Lime's Fort Overhaul
  • Also install Legion Fort Open
Expands the Legion’s Fort base in a lore-friendly style whilst making it a single large area like Simple Open Strip and Freeside.

Desert Bridges Redesigned

Installation instructions:

  • Main File - Improved Desert Overpass
Replaces one ugly, odd looking overpass with a normal-looking one.

Primm Restored

Installation instructions:

  • Main File - Primm Restored
Updates the residents of Primm so they have actual homes, go to them as expected and move around the casino and shop like real town residents would.

Sierra Madre Grand Entrance

Installation instructions:

  • Main File - Sierra Madre Grand Entrance
  • Install the Gate Fix
  • If you installed Desert Natural Weathers in Lighting: install the Desert Natural Weathers patch
Blood Money spends a lot of time building up anticipation to reach an imposing building. Then when you finally reach it… You’re anticlimactically teleported inside. This excellent mod precedes that with a tough combat encounter outside its entrance.
