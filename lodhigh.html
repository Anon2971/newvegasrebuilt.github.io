Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input LOD into the pop-up and select OK

FNVLODGen

Files to install:

  • Main File - FNVLODGen Resources
  • Optional File - Trees LOD Billboards Vanilla

- Resources required for proper LOD generation

LOD Additions and Improvements

Hide the following:

  • textures/clutter/billboard01blod.dds
  • textures/clutter/billboard01blod_n.dds

Manual editing required:

  • Move the meshes/architecture/primm/ folder to textures/architecture/

Manual FNVEdit editing required:

  • Run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to tmzlodadditions.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Input DA726 into the FormID search box in the top left and hit enter
  • Drag and drop NVDefaultExterior from FalloutNV.esm on to WastelandBaseImageSpace from tmzLODadditions.esp
  • Save and exit

- New low-poly models for FNVLODGen

Much Needed LOD

Hide the following:

  • meshes/scol/
  • textures/scol/
  • meshes/landscape/nv_rocks/nvn_rockcanyon19_lod.nif
  • meshes/landscape/nv_rocks/nvn_rockcanyon19blk_lod.nif
  • meshes/landscape/nv_rocks/nvn_rockcanyon19rdrk_lod.nif

Manual FNVEdit editing required:

  • Run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to MuchNeededLOD.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Input 119878 into the FormID search box in the top left and hit enter
  • In the right pane, right-click on the Visible when distant flag and select Remove
    • Repeat for 120F46, 12FC67, and 144B9B
  • Save and exit

- Adds new distant LOD to vehicles, roads, barricades, rocks/cliffs, etc.

ETL - Terrain LOD Noise

Install normally

- Improves the generic LOD noise texture. Edited to reduce the terrainnoise.dds and noisemap.dds to 1k and the noisenormal_n.dds to 512px to reduce shimmering. Original mod by tgspy

Running FNVLODGen

  • Right-click in the left pane of MO2 and select All Mods -> Create empty mod
  • Input FNVLODGen Output into the pop-up and select OK
  • Activate the mod you just created in the left pane of MO2
  • Click the button on the top bar of MO2
  • Select the FNVLODGen option
  • Check the box next to Create Files in Mod instead of Overwrite (*) and from the drop-down menu select FNVLODGen Output
  • Select Modify and then exit
  • Run FNVLODGen through MO2
  • Configure your settings as follows:
    • Each option for Setting for (LOD4, LOD8, LOD16, LOD32) is its own section that needs to be configured
    • If the settings that are grayed-out are different for you, just ignore them
×
  • Once the settings are configured, select Generate
  • Let the generation process run (took around 1.5 hours on my i5) and close the program once the message LOD Generator finished (you can close this application now) appears
