NPC Expansion

Collection 5/10: NPC Expansion

These mods combined fill the Mojave Desert with many more NPCs to interact with.

Mojave Wildlife

Installation instructions:

  • Main File - Mojave Wildlife - Vanilla Version
Adds hundreds more leveled, vanilla-friendly creature spawn points throughout the whole Mojave, based off unused vanilla leveled lists. All spawn points are meticulously hand-placed and distributed as evenly and fairly as possible.

Mojave Raiders

Installation instructions:

  • Main File - Mojave Raiders
  • Additionally install JSawyer Ultimate - Mojave Raiders Patch and Mojave Raiders - Essential Vanilla Enhancements Merged Patch (select the name that matches the file on Nexus when the installation prompt comes up)
  • Additionally install Great Khans' Toughness Increased - Mojave Raiders Patch under Optional Files
Major overhaul/balancing of the vanilla raiders, as well adding many new spawn points.

The Living Desert

Installation instructions:

  • Main File - The Living Desert - Main File 2.3
  • Additionally install The Living Desert - Essential Vanilla Enhancements Merged Patch (select the name that matches the file on Nexus when the installation prompt comes up)
Adds hundreds of new NPCs and scripted events that make the world feel much more alive and reactive.

Faction Wasteland Presence - Improved Edition

Installation instructions:

  • Main File - Faction Wasteland Presence - Improved Edition
Adds Great Khans, Securitrons, NCR and Legion patrols and Brotherhood of Steel paladins on lore-friendly locations.

Brahmin and Big Horner Population and health Increase

Installation instructions:

  • Optional File - Brahmin and bighorner Mod
You’ll come across cattle-like creatures that can get aggressive if you approach them across the Wasteland, but they’re mainly a low challenge food source. This mod increases their quantity whilst also dramatically increasing their health so they’re an actual threat. An excellent survivalist balance tweak.

Lively Merge

Installation instructions:

  • Main File - Lively Merge
  • Optional/contextual - also install the The Bazar of Z - Strip NPCs Uncut - Don't Touch My Street Vendor patch so it functions as expected with Strip NPCs Uncut if you installed that within Restored Content
Adds various lively NPC behaviour AI packages for added immersion.
