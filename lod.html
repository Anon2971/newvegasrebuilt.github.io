LOD Generating Guide

Before getting started, there are a few things you should know. First, you do not need to generate LOD if you just followed the guide. These instructions are only for if you installed texture mods (such as NMCs or Poco Bueno) or are using a mod that requires LOD to be generated (like MoreMojave). Terrain LOD creates a much larger file size (+500mb) and potential performance loss for a very minimal visual improvement. Therefore, this guide will focus only on generating object LOD with xLODGen. If you need to generate xLODGen terrain LOD for whatever reason, the instructions will be the exact same except you will need to enable Terrain LOD option in addition to the Object LOD option.

Installing xLODGen
  1. Download the latest xLODGen file from here
  2. Once the download has finished, extract the contents of the archive to any location of your choice
  3. In MO2, select the button at the top of MO2 to open the executables menu
  4. Click the + at the top of the window and the select Add from file...
  5. From the pop-up, navigate to where xLODGen is installed and select xLODGen.exe
  6. In the Arguments box in the right pane, enter -FNV -O:"C:\Output\"
This argument will make the files generated through xLODGen appear in a folder called Output on your C:\ drive (the drive Windows is installed on). You can change this path to whatever you would like if you would prefer the files to be generated elsewhere. (Not a UAC protected folder or the game folder)
  1. Select Apply then OK

Installing LOD Resources
Make sure you disable Vanilla LOD before proceeding. The plugins from the following mods should be placed directly above DNWeathers.esp, or the plugin of whatever weather mod you are using.
  • FNVLODGen - Main File- FNVLODGen Resources
    • If you are using NMCs Texture Pack, also install the Optional File - NMC LOD texture fix file
  • LOD Additions and Improvements - Main File - LODadditions
  • Only if you use NMCs Texture Pack: Much Needed LOD - JUST the Optional File - Much Needed LOD - NMC Patch
  • LOD Supplementation - Main File - FNV LOD Supplementation
  • LOD Texture Patches - Main File - NMC LOD Patch and/or Main File - Poco-Ojo LOD Texture Patch (depending on which texture mods you are using)
  • TCM's LOD Overhaul - Main File - TCM's LOD Overhaul

Running xLODGen
  1. Select xLODGen from the executables drop-down in the top right of MO2 and click Run
  2. Once the program has loaded, right-click in the left pane and click Select All
  3. In the right pane, check Objects LOD but leave Trees LOD unchecked
Tree LOD is not functional in New Vegas. LOD for trees is covered in Object LOD.
  1. Under the Objects LOD section, change the Atlas size option to 8192 x 8192
  2. Select Generate
  3. Wait for the message LOD Generator: finished (you can close this application now) to appear (generation time depends on your CPU performance), then close out of the program
  4. In MO2, right-click on the Overwrite folder at the bottom of the left pane and select All Mods -> Create empty mod above
  5. Title the mod FNVLODGen Output
  6. Navigate to where the LOD files were generated and select both the textures and meshes folders
  7. Cut them to the FNVLODGen Output mod you just created (right-click on the mod and select Open in Explorer)
